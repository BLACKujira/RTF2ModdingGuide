## Chapter 4.1：PAK Mod

*PAK Mod* 是蓝图Mod的一个超集，它通常不具备扩展游戏系统的功能。其中有两种类型，使用 *UAssetGUI* + *UnrealPak* 制作不需要烘焙的 *PAK Mod* ,这种通常用于修改数据。或者使用 *虚幻引擎* 制作需要烘焙的 *PAK Mod*，这种通常用于替换音乐或贴图。

- 示例：替换背景音乐（需要烘焙）
- 示例：修改武器数据（不需要烘焙）
